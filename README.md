# CodeCrafter AI

## 📌 Project Description

This project is **CodeCrafter AI**, an AI-based Code Generator that takes a programming problem as user input and automatically generates optimized, executable code in the requested programming language (defaults to Python).
The system uses CrewAI agents to understand the problem and produce correct solutions following best practices and optimal time complexity.

 ## 🎯 What This Project Does

Accepts programming tasks as input

Generates ready-to-run code in your language of choice

Produces optimized solutions (O(1), O(n), O(log n) where applicable)

Supports DSA and system-level problems

Automates code writing using agent-based AI

## ⚙️ Tech Stack

Python 3

CrewAI

Groq LLaMA 3.1

Virtual Environment (venv)

## ▶️ How to Run

Activate virtual environment: venv\Scripts\activate

Run the project: python crew.py
